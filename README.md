# Minesweeper

A complete terminal-based Minesweeper game built in Kotlin as a university project for Fundamentos de Programação course at Universidade Lusófona (2025/2026).

## Features

- Full Minesweeper game logic
- Configurable board size and mine count
- Input validation
- Recursive cell reveal
- Menu system

## Technical Constraints

This project was built under strict academic constraints to deepen understanding of fundamental programming constructs:

- No high-level data structures (lists, maps, sets)
- No `for` loops, only `while`
- No `break`, `continue`, or `exitProcess`
- No external libraries beyond Kotlin's standard library

## How to Run

```bash
kotlinc Main.kt -include-runtime -d Minesweeper.jar
java -jar Minesweeper.jar
```

*(Adjust the command if your project structure is different.)*

Built collaboratively in a 2-person team, with individual defense after submission.
